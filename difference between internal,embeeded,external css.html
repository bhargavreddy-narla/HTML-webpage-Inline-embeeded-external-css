Diff between Inline, Embeeded, External CSS
INLINE CSS EMBEEDED CSS EXTERNAL CSS
  • Configured in the body of the web page
  • Configured in the header section of a web page
  • Configured in a seperate text file with .CSS file extention
  • Use the style attribute of an XHTML tag
  • Use the XHTML style element
  • The XHTML link element in the header section of a web page associates it with the .css file
  • Apply only to the specific element
  • Apply to the entire web page document
